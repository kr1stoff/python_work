while语句
for循环用于针对集合中的每个元素都一个代码块，而while循环不断地运行，直到指定的条件不满足为止。
你可以使用while循环来数数

?  prompt = "\nTell me something, and I will repeat it back to you:"
　　prompt += "\nEnter 'quit' to end the program. "
?  message = ""
?  while message != 'quit':
　　　　message = input(prompt)
　　　　print(message)
我们将变量massage的初始值设置为空字符串””，让python首次执行while代码时有可供检查的东西，首次执行while时，需要将message的值与’quit’比对，如果没有可供比对的东西，python无法运行。
要求很多条件都满足才继续进行的程序中，可定义一个变量，用于判断整个程序是否处于活动状态，这个变量被称为标志。
while循环中，我们输入一条if语句来检查message的值，如果用户输入的是’quit’，我们将标志设置为False，这将导致while循环不再继续进行。
要立即退出while循环，可使用break语句，使用它控制那些代码执行那些代码不执行。
当while True：时循环将一直执行，直到break
continue语句不像break语句不再执行整个循环，而是满足if条件的值忽略后面的代码，从循开头继续执行循环
避免无限循环，要避免编写无限循环，无比对每个while语句进行测试
